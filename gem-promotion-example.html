ทดสอบกับวันที่กำหนดเอง

วิธีการใช้งาน

1. เรียกใช้ผ่าน CDN:

<script src="path/to/gem-promotion-cdn.js"></script>

2. สร้าง Full Promotion (ปกติ):

<div id="my-promotion"></div>
<script>
  GemPromotion.createFullPromotion('my-promotion', 'GEMS2025', 'ส่วนลด 20%');
</script>

3. กำหนดวันที่เอง (สำหรับทดสอบ):

<div id="test-promotion"></div>
<script>
  GemPromotion.createFullPromotion('test-promotion', 'TEST2024', 'ส่วนลดทดสอบ 50%', {
    eventStart: '7/12/2024 10:00',   // วัน/เดือน/ปี ชั่วโมง:นาที
    eventEnd: '8/12/2024 18:00',     // วัน/เดือน/ปี ชั่วโมง:นาที
    title: 'งานทดสอบโปรโมชั่น',
    dateText: '7-8 ธันวาคม 2024'
  });
</script>

4. รูปแบบวันที่ที่รองรับ:

// รูปแบบที่ใช้ได้:
'7/12/2024'          // วัน/เดือน/ปี (เวลา 00:00)
'7/12/2024 14:30'    // วัน/เดือน/ปี ชั่วโมง:นาที
'2024-12-07'         // ปี-เดือน-วัน (เวลา 00:00)
'2024-12-07 14:30'   // ปี-เดือน-วัน ชั่วโมง:นาที

5. ตัวอย่างการทดสอบในปัจจุบัน:

// ทดสอบให้โปรโมชั่นเปิดใช้งานได้ทันที
const now = new Date();
const start = new Date(now.getTime() - 60000); // เริ่ม 1 นาทีที่แล้ว
const end = new Date(now.getTime() + 3600000);  // สิ้นสุดใน 1 ชั่วโมง

GemPromotion.createFullPromotion('test', 'ACTIVE', 'ใช้ได้เลย!', {
  eventStart: start,
  eventEnd: end
});

คุณสมบัติ:

  • นับถอยหลังไปยังวันงานที่กำหนด
  • รหัสส่วนลดซ่อนก่อนวันงาน แสดงเป็น ••••••••
  • คัดลอกได้เฉพาะช่วงวันงาน
  • กำหนดวันที่เอง สำหรับทดสอบ
  • Responsive design ใช้งานได้บนมือถือ
  • ไม่ต้องการ CSS เพิ่มเติม
